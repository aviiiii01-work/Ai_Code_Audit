# Bootcamp - Full Week-wise Solutions

## Overview
This repository consolidates all the week-wise tasks, projects, and exercises from the MERN & Development Bootcamp. Each week is organized into its own subfolder for easier navigation and reference.

## Repository Structure

```
bootcamp/
│
├── bootcamp_week1/   # All Day 1 to Day 5 tasks of Week 1
├── bootcamp_week2/   # All Day 1 to Day 5 tasks of Week 2
├── bootcamp_week3/   # All Day 1 to Day 5 tasks of Week 3
└── README.md                   # This file
```

### Week-wise Breakdown

- **Week 1:**  
  - Contains Day 1 to Day 5 tasks covering basic Node.js, HTML/CSS, JavaScript fundamentals, and introductory backend exercises.  
- **Week 2:**  
  - Includes Day 1 to Day 5 tasks, focusing on frontend development projects, intermediate JavaScript, and backend interaction using Node.js.  
- **Week 3:**  
  - Advanced topics including full-stack integration, API handling, and project-based assignments.  

*(Each week's folder contains individual day folders with their respective files, solutions, and README documentation for clarity.)*

## How to Use

1. Clone this repository:

```bash
git clone https://github.com/RaviPratapSinghHestabit/BOOTCAMP.git
```

2. Navigate to the desired week and day folder:

```bash
cd BOOTCAMP/bootcamp_week1/bootcamp_day1.git
```

3. Open files in your favorite code editor (VSCode recommended) and follow the README in each day folder for instructions.

4. Run any Node.js projects using:

```bash
node server.js
```

*(Follow the day-specific README for details on testing endpoints and running scripts.)*

## Notes

- Each week’s folder is self-contained and can be studied or executed independently.
- All projects include README files detailing the task, approach, and instructions.
- Git has been structured to avoid nested repositories; everything is tracked under this main repo.
